Predicting Hypertension

Introduction

Hypertension is a condition where the pressure in a persons arteries is consistently abnormally high, which can lead to many fatal risks such as stroke, kidney diseases, and other cardiovascular diseases (Baliga and Bakris, 1). Studies have shown that unhealthy levels (Baliga and Bakris, 29) of sodium and above-average glucose concentrations are common among hypertensive men.

The dataset used in this project is from UC Irvines Machine Learning Repository and contains 25 different variables. It has been collected from a hospital within a two month period, and is designed for predicting chronic kidney disease. In this project, we wanted to determine whether leading factors of hypertensions explored by several medical research articles are also leading factors for hypertension in our data set taken from the UC Irvines Machine Learning repository collected for Chronic Kidney Disease. That is, we wanted to examine if the factors: sodium, blood pressure, and age are good predictors for hypertension. We define good as predictors with an accuracy of above 85% in their respective classifiers, and classifiers without these predictors do not fluctuate more than 5% in accuracy

KNN Analysis

We conducted a k nearest neighbours analysis. The k-value was selected after we tested hyperparameter values using the seq() function; we selected a k-value that gives the highest accuracy (within a reasonable size). To train the model, we decided to use a 10-fold cross validation that would help select an optimal k value. In regard to splitting our data into a certain number of observations, we partitioned it in a way that we will have 25% as test set and 75% as training set to train our classifier. Each classifier had different number of observations left before they were partitioned; this is because when we removed the rows that had contained NA values, the number of observations had been reduced. For example, the first classifier had 297 rows left before it was partitioned, whereas the second classifier had 259 rows left before it was partitioned. Moreover, the KNN algorithm calculates the Euclidean distance between all the data points; the data is already scaled.

Model

We built four different models, each using three different variables to predict whether a person has hypertension or not. The first classifier is the control, where the predictors age, blood pressure, and sodium levels are used.The other three models are each run using three randomly selected predictors. To randomize variables, we took all the existing numerical variables in the data set, and randomized them using an online list randomizer from random.org. By comparing these classifiers, we can assess our hypothesis.

Classifier 1

The first classifier predicts based on the three leading predictors (Age, Blood Pressure and Sodium Levels) from our research on hypertension. We only chose numerical data points from the CKD dataset. We also decided to plot different values of k against its training accuracies in order to pick the smallest k with highest accuracy and then proceeded to apply our model on our testing set.

Also, for reproducibility we set the seed to 1234 each time we trained our model.

Upon performing the k-nn algorithm analysis, we discovered that the first classifier instead experienced significant drop in accuracy. The first classifier had an accuracy of 79.73%, whereas the other three classifiers had an average accuracy of 85.73%. Since the first classifier had and accuracy rate of less than 85%, the three factors we selected were not good predictors. We believe a potential cause of this discrepancy is from the data set we used. It was intended for dealing with Chronic Kidney Disease, instead of hypterension; moreover, the medical journals may have used a data set specifically designed for Hypertension. Overall, the difference in the type of datasets used could have contributed to the results.

Such findings could potentially cause other classification algorithms to be used for hypertension, as the result of a k-nn classification does not parallel that of existing research. This could potentially lead to questions about which algorithm would be optimal for this classification. Moreover, further analysis should be conducted on a hypertension specific dataset, to isolate whether the algorithm or the data is the cause of the differences.
